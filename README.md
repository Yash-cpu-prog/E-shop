# 🛒 E-shop - Full Stack Ecommerce Website

A modern full-stack ecommerce web application built using the MERN stack (MongoDB, Express, React, Node.js).  
This project provides a seamless shopping experience with authentication, product listing, cart system, and order management.

---

## 🚀 Features

- 👤 User Authentication (Login / Register)
- 🛍️ Product Listing & Categories
- 🔍 Product Search & Filter
- 🛒 Add to Cart / Remove from Cart
- 💳 Checkout System
- 📦 Order Management
- 🔐 Secure Backend APIs
- 📱 Fully Responsive UI

---

## 🧑‍💻 Tech Stack

### Frontend:
- React.js
- Redux / Context API
- Tailwind CSS / CSS

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

### Other Tools:
- JWT Authentication
- Bcrypt.js
- REST API
